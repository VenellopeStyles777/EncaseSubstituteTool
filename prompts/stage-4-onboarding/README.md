# Stage 4 Onboarding

Purpose: handoff packet for the next research/review agent before Stage 4 hash/signature work begins.

This folder is separate from `prompts/vscode-agent/`. The review-agent reflection here is from the Stage 3 research/review perspective: risks, review gates, missing evidence-backed paths, and future-ticket scrutiny. The coding-agent handoff/reflection belongs in `prompts/vscode-agent/` and may emphasize implementation mechanics differently.

Use these files before writing Stage 4 tickets or prompts:

- `stage-4-review-agent-familiarization-prompt.md`: paste-ready prompt for the Stage 4 review agent.
- `project-reflection-and-forward-risks.md`: Stage 3 closing reflection, weakest-point assessment, and forward-stage warnings.

Stage 4 review work should begin with a whole-project familiarization pass. Do not merge this packet with the coding-agent onboarding prompt.
