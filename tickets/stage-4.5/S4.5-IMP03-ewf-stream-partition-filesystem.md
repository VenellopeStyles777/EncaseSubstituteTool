## Objective

Implement the first reviewed bridge from an E01 segment set toward navigable forensic structure: an EWF-backed image stream boundary, partition discovery statuses, and minimal root filesystem metadata/listing when dependencies and parser support are available.

This ticket must preserve dependency-free defaults and honest unavailable/not-implemented states.
